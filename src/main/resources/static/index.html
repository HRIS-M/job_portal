    Want to report an issue?

    Here is some Troubleshooting Advice for various scenarios:

    1. Account details are incorrect
      • Try to login with correct credentials.
      • Reload the page. If problem persists, contact us.
      • This error can be happen when you using multiple accounts with same device.
    2. Saved job details not visible
      • This error can happen when you are not logged in correctly.
      • The company/job post might inactive or deleted/closed.
    3. Failed to apply a job
      • Check you're logged as a candidate.
      • If the problem is still persists, contact us.
    4. Failed to post a job
      • Check you're logged as an employer.
      • Ensure required information provided in organization profile.
      • Check your job posting limit exceeded. (you can post new jobs after removing expired jobs)
      • If the problem is still persists, contact us.
    5. Failed to upload multimedia files (images, videos, documents, CVs)
      • Make sure files are not reached the limit.
      • We allow *.jpeg, *.jpg, *.png, *.pdf file formats only.(some cases allows *.gif, *.doc, *.docx)
    6. File uploaded. But not updated the content
      • Reload page. Sometimes cache persists and failed to update without reloading.
      • Re-upload the file. When you are tring to upload a new file, sometimes first attempt create a specific path for only you. So try again once:)
      • If the problem is still persists, contact us.
